Cloud Sync (Autosave)

Everything you type in this manager (clients, invoices, contracts, tasks, pipeline, productions, calendar events) is autosaved to your PRIVATE GitHub repo every few seconds after each edit. Open the manager on any device, log in, paste your GitHub PAT above, and your data loads.
File: SHIFT_DATABASE_V2.json in produccion-cmd/shift-data (private). Only your PAT can read/write it. Not synced: password, PAT itself.
